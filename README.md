# EP2-IA---Plantio

O nosso programa foi feito para resolver o problema proposto de plantio rural.

Na pasta docs você pode encontrar os dois arquivos em markdown detalhando o nosso programa e fazendo um estudo de parâmetros.
Na pasta src voce pode encontrar o nosso programa .py onde você pode executar alguns parâmetros, ou apenas o programa onde serão executados alguns valores padrão.

Exemplo de comando:
python EP2.py --pop 100 --gen 200 --mut 0.2
